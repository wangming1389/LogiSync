# Mobile Docs

Khu vực dành cho tài liệu `apps/mobile`.

## Quick Start

Mobile app hiện là Expo app.

```bash
pnpm --filter @logisync/mobile dev
```

Nếu cần reset starter scaffold:

```bash
pnpm --filter @logisync/mobile reset-project
```
